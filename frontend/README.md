# Фронтенд журнала работ (React + TypeScript)

Интерфейс «Журнала работ» для строительного контроля, разработанный на React, TypeScript и Tailwind CSS с использованием TanStack Query для взаимодействия с бэкенд API.

## Используемые библиотеки
*   `react` + `typescript`
*   `framer-motion` (плавная анимация интерфейса)
*   `@tanstack/react-query` (клиентское кэширование и реактивный стейт-менеджмент)
*   `lucide-react` (минималистичные строительные и системные иконки)

## Запуск в режиме разработки
Для запуска фронтенда отдельно в режиме разработки выполните в этой папке:

```bash
npm install
npm run dev
```

Интерфейс запустится локально и будет доступен в браузере. Все запросы к `/api` будут автоматически пересылаться на локальный бэкенд NestJS на порту `3001` благодаря настроенному прокси в `vite.config.ts`.
